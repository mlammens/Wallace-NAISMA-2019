Fixing or Cleaning Data

Lets say we realized that we had a mistake in our data. For example, one of UNCERTAIN values was recorded incorrectly. How can we change this?

fral_pres_fixed <- fral_pres
fral_pres_fixed$UNCERTAIN[1]
## [1] NA
fral_pres_fixed$UNCERTAIN[1] <- 20
fral_pres_fixed$UNCERTAIN[1]
## [1] 20

Subset the data

Lets get only a subset of these data, selecting from the data.frame by columns.

fral_pres_subset <- fral_pres[c("SPEC", "LONG", "LAT")]

Next, lets rename our columns so they are in the format used in Wallace.

names(fral_pres_subset) 
## [1] "SPEC" "LONG" "LAT"
names(fral_pres_subset) <- c("name", "longitude", "latitude")

Lets now make a new file with the fixed data.

write.csv(x = fral_pres_subset, file = "~/Dropbox/SCCS-Workshop/fral_pres.csv", row.names = FALSE)
